# King of Snakes

King of Snakes is a jungle-themed snake web game built with HTML, CSS, and JavaScript.

## Features
- Desktop and mobile browser support
- Touch controls for mobile
- Golden bird with magical eggs
- Pink bird on even levels
- Permanent wood obstacles
- Win screen at Level 50
- Background music and sound effects
- Local high score saving

## How to Play
- On desktop: press **Enter** or click/tap to start
- On mobile: tap to start
- Controls:
- Desktop: Arrow keys
- Mobile: Swipe or on-screen controls
- Eat white eggs to grow and score
- Golden eggs reduce speed and length
- Pink egg gives a level jump bonus
- Avoid walls, your own body, and wood obstacles

## Files
- `index.html` — main page
- `style.css` — game styling
- `script.js` — game logic
- `snake-title.png` — optional title image
